# Url_Input Control

The `Url_Input` control is a `Text_Input` with `type="url"`.

## Usage

```javascript
const url_input = new controls.Url_Input({
    context,
    value: 'https://example.com'
});
```

## Notes

- Inherits behavior from `Text_Input`.
